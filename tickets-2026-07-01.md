> ✅ Resolved 2026-07-02 — Permanent 308 redirect added to vercel.json with has-host rule for cinematen.be → www.cinematen.be.

**🟪 Wat / What**

**💬 Impact**  
HTTP 307 signaleert aan browsers en bots dat de redirect tijdelijk is en kan veranderen — link equity wordt niet doorgegeven. Met 92 externe backlinks (allemaal naar podcast-directories) is elke link-equity-lekkage significant. 301/308 geeft volledige authority-overdracht.

**🚦 Severity**  
🔴 Blocking

**🔍 Details**
- **Area:** Vercel config / redirects
- **Affected files:**
  - `vercel.json` — voeg toe aan `redirects` array (regel 49):
    ```json
    {
      "source": "/:path*",
      "destination": "https://www.cinematen.be/:path*",
      "permanent": true,
      "has": [{ "type": "host", "value": "cinematen.be" }]
    }
    ```
- **Effort:** ~5 min
- **Priority:** 🔴 Kritiek — Week 1
- **Notes:** `permanent: true` in Vercel JSON = HTTP 308. Kan ook via Vercel Dashboard → Project Settings → Domains. Test na deployment met `curl -I https://cinematen.be` — verwacht `308`.
